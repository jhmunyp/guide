비밀번호 찾기 안내

아래 ‘단계’를 그대로 따라하면 비밀번호를 바꿀 수 있어요.

천천히 따라해 주세요

  • 각 단계의 화면과 같이 진행하면 됩니다.
  • 화면이 작으면 이미지를 눌러 크게 보기가 가능합니다.
  • 진행이 어렵다면 아래 고객센터 전화 버튼을 눌러주세요.
주의 인증번호는 고객센터/직원이 요구하지 않습니다. 전화·문자 사칭에 주의하세요.
단계 1

로그인 화면에서 ‘비밀번호 찾기’

로그인 화면에서 비밀번호 찾기를 눌러주세요.

단계 2

비밀번호찾기 화면에서 인증 방식 선택

아이디 입력 후 휴대폰 인증 또는 아이핀(IPIN) 인증을 선택합니다.

단계 3

NICE 본인인증에서 인증 수단 선택

휴대폰 본인 확인 또는 공동인증서 중 선택합니다.

단계 4

통신사 선택

이용 중인 통신사를 선택합니다. (알뜰폰도 선택 가능)

단계 5

인증 방법 선택

PASS, QR코드, 문자(SMS) 중 선택합니다.

단계 6

문자(SMS) 인증 선택 후 ‘동의(필수)’ 체크

아래 본인확인 이용 동의(필수)를 체크하고 다음을 누릅니다.

단계 7

정보 입력

이름/생년월일/휴대폰번호/보안문자를 입력하고 확인을 누릅니다.

단계 8

휴대폰 문자로 인증번호 받기

휴대폰으로 온 인증번호(6자리)를 확인합니다.

단계 9

인증번호 입력

받은 인증번호 6자리를 입력하고 확인을 누릅니다.

단계 10

본인 인증 완료

인증이 완료되면 비밀번호 변경 화면으로 넘어갑니다.

단계 11

새 비밀번호 입력 후 변경

새 비밀번호와 확인을 입력하고 변경을 누릅니다.

단계 12

완료

이제 로그인 화면에서 새 비밀번호로 로그인해 주세요.

📞 고객센터 전화 도움이 필요하시면 눌러주세요 ⬆︎

이미지

팁: 확대된 화면은 손가락으로 확대(핀치)할 수 있어요.
